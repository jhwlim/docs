공부한 내용을 정리한 블로그 입니다.

블로그 주소 : <https://jhwlim.github.io/>

## 로컬에서 실행하기

`_config.yml`을 수정하고,

```yml
# Build settings
theme: "just-the-docs" # 주석을 해제한다.
remote_theme: pdmosses/just-the-docs@callouts
```

<br>

터미널에서 아래의 명령어를 실행한다.

```shell
jekyll serve
```

또는

```shell
bundle exec jekyll serve
```
